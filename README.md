# service2 — мини-сервис с регистрацией и перемешиванием строк

Небольшой локальный проект, на котором я разбирался, как связать бэкенд на FastAPI с простым фронтендом без использования тяжелых фреймворков.

### Что делает приложение:
1. **Первый шаг (Регистрация)**: Пользователь вводит имя и возраст. На бэкенде через Pydantic стоит проверка — если юзеру меньше 18 лет, сервер выдает ошибку и не пускает дальше. Если все ок, данные дописываются в текстовый файл `users_data.txt`.
2. **Второй шаг (Перемешивание)**: Открывается скрытая форма. Вводишь любую строку (например, `хкуцлкуцло`), жмешь кнопку — сервер случайно перемешивает буквы между собой и возвращает результат обратно на страницу.

### Из чего состоит проект:
* **app/schemas.py** — схемы данных и кастомный валидатор возраста.
* **app/config.py** — настройки CORS, чтобы браузер не блокировал запросы к локальному серверу.
* **app/main.py** — логика эндпоинтов (запись в файл и функция перемешивания символов).
* **index.html** — простая SPA-страница на чистом JS (Fetch API) с обработкой ошибок от бэкенда.
* **Dockerfile** — конфигурация для сборки проекта в контейнер.

---

### Как запустить проект у себя:

Так как в Windows стандартные команды Python могут капризничать, надежнее всего запускать через встроенный загрузчик `py`.

1. Клонируем или скачиваем репозиторий, заходим в папку проекта через терминал.
2. Устанавливаем необходимые библиотеки:
   ```bash
   py -m pip install -r requirements.txt
   ```
3. Запускаем локальный сервер Uvicorn:
   ```bash
   py -m uvicorn app.main:app --reload --port 8000
   ```
4. Просто открываем файл `index.html` двойным кликом в любом браузере и тестируем.
